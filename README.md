# 🛒 E-Mart – Modern E-commerce Platform

## 📌 Project Purpose

**E-Mart** is a modern **MERN stack e-commerce platform** designed to provide a smooth and scalable online shopping experience for customers and administrators.

The goal of this project is to build a **secure, responsive, and user-friendly online marketplace** where users can browse products, manage carts, add items to wishlists, and place orders seamlessly.

This project demonstrates **real-world full-stack development skills**, including authentication, role-based dashboards, product management, and API integration.

# ⚙️ How the System Works

1. **Users** can browse all available products from the homepage.
2. Users can view detailed information for each product.
3. Logged-in users can add products to **cart** or **wishlist**.
4. Users can manage items from the cart page.
5. Admin users can **manage products and platform data** from the dashboard.
